# Projet diplome

LisibilitéCe projet propose une expérience interactive mêlant site web ludique et projection immersive. L’utilisateur retourne des cartes façon jeu de mémoire pour trouver des paires. Chaque paire permet de lancer une énigme en projection : sa silhouette, captée par caméra, révèle une image cachée. Deux mots mènent à un troisième, saisi au clavier.
